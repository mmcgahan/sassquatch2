Attachment

Adding the attachment class to an elemeent creates a context in which layout classes (row-item, chunk, list) use half of their default spacing.

In foundation, an "attachment" is a piece of content in the feed attached to a parent item (ie. a reply or event meta info).

Ancestry only affects spacing in attachment. Text sizing remains explicit.

Event meta info example

13

Monday Jan 13, 6:30 pm

Backpacking For Beginners

<div class="row attachment chunk">
    <div class="row-item row-item--shrink">
        <time class="tearsheet" datetime="2017-01-13">13</time>
    </div>
    <div class="row-item">
        <h4 class="text--caption">Monday Jan 13, 6:30 pm</h4>
        <h4>Backpacking For Beginners</h4>
    </div>
</div>

Avatar variants

A base class of avatar is required. The following classes are optional variants:

Class Description
avatar--small Adjusts avatar size to the smallest media size
avatar--big Adjusts avatar size to the biggest media size
avatar--person Variant for people
avatar--org Variant for organizers
avatar--fbFriend Variant for facebook friends
avatar--icon For icons in avatars (might move this in a later version)

Button variants

A class of .button is required for buttons. The following classes are optional variants:

Class Description
.button--primary rounded button with dark background; for primary actions
.button--contrast bold, dark text; use on dark backgrounds or photos (ie. top nav bar)
.button--bordered rounded bordered button
.button--small decreases size of button

Tabs (parent) variants

A class of tabs is required. The following classes are optional variants:

Class Description
tabs--full Makes tabs span the full width of their parent element

Tab (child) variants

A class of tabs-tab is required for tabs. The following classes are optional variants:

Class Description
tabs-tab--selected Applies styles for tab selected state

RSVP Tags

Status tags for RSVP.

Going
Not going
Waiting
<div class="rsvpTag rsvpTag--yes">Going</div>
<div class="rsvpTag rsvpTag--no">Not going</div>
<div class="rsvpTag rsvpTag--waiting">Waiting</div>

Variants

A class of rsvpTag is required. Variant classes:

Class Description
rsvpTag--yes indicates "yes" RSVP status
rsvpTag--no indicates "no" RSVP status
rsvpTag--waiting indicates waitlist RSVP status

Toggle Pills

"unchecked" state

    <input class="toggleButton-input" type="checkbox" name="topic-dining" id="topic-dining" value="ToggleBtn" />
    <label class="toggleButton-label" for="topic-dining">Dining Out</label>
</div>

"checked" state

<div class="toggleButton">
    <input checked class="toggleButton-input" type="checkbox" name="topic-hiking" id="topic-hiking" value="ToggleBtn" />
    <label class="toggleButton-label" for="topic-hiking">Hiking</label>
</div>
