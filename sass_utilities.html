Sass Functions

Contrast/brightness utils

Perceived brightness value

Gets brightness of a color on a scale of 0-255 based on a yiq color space. Calculates green-weighted sum of RGB channels (green is perceived as brighter on additive color screens).

returns int

getPerceivedBrightness(#000) // -> 0

Programmatic contrast text

For a given background color, this function returns a text color value with appropriate contrast.

returns color value

// -> color: $C_textPrimaryInverted; (white text on dark)
color: getPrimaryTextColor(#000);

// -> color: $C_textPrimary; (dark text on white)
color: getPrimaryTextColor(#FFF);

isDark/isLight

Checks a given color for brightness and returns a boolean. Our default dark/light threshold is 160 on a scale of 0-255

returns boolean

isDark(#000) // -> true
isLight(#000) // -> false

String utils

str-firstCharToUpper

Capitalizes first letter of a given string. Useful as a camelCase converter.

returns string

str-firstCharToUpper('snowflake') // -> 'Snowflake'

.special#{str-firstCharToUpper('snowflake')} // -> .specialSnowflake

str-replace

Search and replace on strings.

returns string

str-replace('inline-block', '-', '') // -> 'inlineblock'

.display--#{str-replace('inline-block', '-block', 'Block')} -> .display--inlineBlock

Media Query Breakpoints

Mobile first mixin

This breakpoint mixin uses a mobile first strategy, meaning the medium breakpoint, for example, applies to medium screens and larger.

    @include atMediaUp(medium) {
        body::after {
            content: "medium screens and larger can see me";
        }
    }
Breakpoint name Description
atMediaUp(small) Small sreens and up
atMediaUp(medium) Landscape on most phones and up; portrait on some phablets
atMediaUp(large) Landscape on phabelts and up; portrait on most tablets or old desktops
atMediaUp(huge) Modern desktop viewports and up or ridiculous huge tablets

Breaking for smaller sizes and under

In the event you need to target a specific screen size and under, use the atMediaDown mixin.

    @include atMediaDown(medium) {
        body::after {
            content: "medium screens and smaller can see me";
        }
    }
Breakpoint name Description
atMediaDown(small) Small screens and under; portrait on most mobile devices
atMediaDown(medium) Landscape on most phones and under; portrait on some phablets
atMediaDown(large) Landscape on phabelts and under; portrait on most tablets or old desktops
atMediaDown(huge) Huge screens and under

Targeting a specific breakpoint

It will be rare that we target a specific breakpoint, but the atMediaOnly mixin will do this.

    @include atMediaOnly(medium) {
        body::after {
            content: "only medium screens will see this";
        }
    }
Breakpoint name Description
atMediaOnly(small) Small screens only
atMediaOnly(medium) Landscape on most phones only
atMediaOnly(large) Landscape on phabelts only
atMediaOnly(huge) Huge screens only
